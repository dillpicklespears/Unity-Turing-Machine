# Turing Machine Unity

This project is based on creating a working turing machine in Unity.

## Description

This program generates a tape of 0s (red) and 1s (blue). A dancing star represents the head of the turing machine as it slides across the tape. The numbers and colors change upon being written. 

This project will be iterated more on projects 2 and 3.
